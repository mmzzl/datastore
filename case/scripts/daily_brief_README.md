# 每日收盘简报系统

## 功能概述

本系统实现了每日收盘简报功能，包含以下分析维度：

| 维度 | 功能 | 说明 |
|------|------|------|
| 1. 大盘与市场环境 | 整体涨跌、市场情绪 | 计算涨跌家数、平均涨跌幅、涨停跌停数 |
| 2. 板块热点与轮动 | 行业板块涨跌幅 | 基于证监会行业分类，计算板块平均涨跌幅 |
| 4. 个股表现与活跃度 | 涨幅榜、跌幅榜、振幅榜 | TOP20 涨跌幅、振幅、成交额排行 |
| 6. 技术信号与趋势 | MA金叉、RSI超买超卖 | MA5/MA20金叉、RSI>80超买、RSI<20超卖 |

## 使用方法

### 1. 自动获取最新交易日并生成简报（推荐）

```bash
# 自动获取最新交易日，检查数据是否存在，不存在则下载，然后生成简报
py daily_close_brief.py
```

**说明**：
- 自动从 baostock 获取最新交易日
- 检查最新交易日的数据是否存在
- 如果数据不存在，下载最新交易日这一天的数据
- 如果数据已存在，跳过下载，直接分析
- 自动生成最新交易日的收盘简报

**注意**：请确保系统时间正确，否则会获取错误的交易日

### 2. 指定日期生成简报

```bash
# 分析指定日期的简报（会自动下载该日期的数据）
py daily_close_brief.py --analyze_date 2024-01-05
```

### 3. 仅分析已有数据

```bash
# 分析指定日期的数据（默认分析数据中最新的日期）
py scripts\daily_brief_analyzer.py
```

## 数据格式

### 输入数据（CSV 格式）

```
date,symbol,open,high,low,close,volume,amount,adjustflag
2024-01-02,SH600000,6.63,6.65,6.6,6.6,22066700,146066303.72,3
2024-01-03,SH600000,6.59,6.65,6.59,6.64,18203654,120639706.01,3
```

### 计算指标

| 指标 | 计算公式 | 说明 |
|------|----------|------|
| 涨跌幅 | (今日收盘 - 昨日收盘) / 昨日收盘 × 100% | 正数表示上涨，负数表示下跌 |
| 振幅 | (最高价 - 最低价) / 最低价 × 100% | 当日价格波动幅度 |
| MA5 | 最近5日收盘价的移动平均 | 短期趋势指标 |
| MA20 | 最近20日收盘价的移动平均 | 中期趋势指标 |
| RSI | 相对强弱指标 | RSI>80超买，RSI<20超卖 |

## 注意事项

1. **数据来源**：使用 baostock API，免费但有请求频率限制
2. **数据范围**：支持沪深两市 A 股数据
3. **时间范围**：建议每次下载不超过 1 个月的数据
4. **并发设置**：`max_workers=1` 避免并发登录问题
5. **数据延迟**：baostock 数据可能有 1-2 天的延迟

## 常见问题

### Q: 为什么有些股票显示 "is empty"？
A: 可能是股票停牌、退市，或者 baostock API 暂时无数据。

### Q: 涨跌幅计算不准确？
A: 涨跌幅是基于前一日收盘价计算的，如果数据不连续可能导致计算错误。

### Q: 为什么股票数量不对？
A: 由于多次下载数据，CSV 文件中可能存在重复数据。系统已自动去重，每个股票在指定日期只保留一条记录。

### Q: 为什么有 2026 年的数据？
A: 这是测试数据或错误数据。系统已自动过滤掉 2026 年的数据。

### Q: 如何添加自定义指标？
A: 在 `DailyBriefAnalyzer` 类中添加新的计算方法，并在 `print_brief` 中调用。
